## 덤으로 vi 사용해보기
* 파일열기 : vi 파일이름
* insert(편집) mode 진입 : vi 처음화면에서 i
* overwrite(편집) mode 진입 : vi 처음화면에서 o

* 편집>>명령 mode 전환 : esc키(1번 내지는 2번을 눌러주어야할때도 있음)
* vi 종료하기 :q
* 편집내용 저장하기 :W
* 편집내용 저장하지 않고 끝내기 :q!
* 편집한 내용 저장하고 끝내기 :wq
